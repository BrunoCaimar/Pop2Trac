Pop2Trac - Initial version
Get emails to create tickets on Trac using email2trac.py 
http://github.com/BrunoCaimar/Pop2Trac

This is partially based on Unfuddle Pop2Ticket.NET - http://pop2ticket.codeplex.com/
It's use email2trac.py (https://subtrac.sara.nl/oss/email2trac) to parse and create tickets in Trac

The flow to use it is:
- Adjust config file (Pop2Trac.config and email2trac.conf)
- Run Pop2Trac.exe (It'll get the emails on server and run email2trac.py to parse/create ticket on trac)

Dependencies:
- Python
- email2trac.py package
